# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-05-26

### Added
- Initial release of the Student Result Management System.
- Feature to upload and parse Excel result sheets.
- Automatic calculation of subject marks, totals, and percentages.
- Interactive CLI menu using `curses`.
- Data visualization support (Bar, Line, Scatter charts).
- Grouping of students based on percentage ranges.
- Data persistence using local CSV files in `user-data/`.
- Export functionality to Excel.
